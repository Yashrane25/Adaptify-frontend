# Adaptify-frontend

## Adaptify

### 🚀 Where Innovation Meets Inclusion

Adaptify is an advanced accessibility-focused system that enables effortless interaction through eye-tracking, gesture control, and speech-to-text technology. Designed to make the digital world accessible for everyone.


## 🌟 Features

Eye Tracking: Navigate and control using just your eyes.

Gesture Control: Hands-free interaction through intuitive gestures.

Speech-to-Text: Convert voice into text for seamless communication.

User-Friendly Interface: Simple, elegant, and accessible UI.
